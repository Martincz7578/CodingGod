Coding God
---

A simple game in TypeScript, featuring base buidling, well only that for now. The game might change a lot.

Saving has not been implemented, but will be very soon.

To play this game you need PHP 8.4 and TS 5.9.

How to run
---
<ol>
    <li>run <code>tsc</code></li>
    <li>run <code>php -S localhost:9000</code></li>
    <li>open <code>localhost:9000</code> in your browser</li>
</ol>

Controls
---
B - Building
Space- place (Might change later)
C - Clear

And that's it! Enjoy!
